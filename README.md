# Advent of Code 2024

My solutions to the Advent of Code 2024. Written in PHP.

## Installation

```
composer install
```

## Running test

```
./vendor/bin/pest
./vendor/bin/phpunit
composer test
composer pest
```

## Running opinionated PHP code style fixer for minimalists

```
./vendor/bin/pint 
composer lint
```

## Running a solution

```
composer solve <day> <puzzle>
```

## Running all solutions

```
composer solve all
```

## Generating the next day

```
bash ./scaffold.sh <day>
```
